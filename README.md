# lnx_hw_simple_project


## Installation

1. Clone the repository:
    git clone https://github.com/KilledByMachine/lnx_hw_simple_project
    cd your_repo

2. Build the program:
    ./configure
    make

3. Install the program:
    sudo make install

## Use

After installation, you can use the program by simply running it:

myprogram
